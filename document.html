Video.js Multi-Audio & Subtitles - Documentation

1. Built-in Audio Tracks

Video.js detects multiple audio tracks in HLS streams, allowing users to switch dynamically. Use player.audioTracks() to access and display available options.

var audioTracks = player.audioTracks();
for (let i = 0; i < audioTracks.length; i++) {
    console.log(audioTracks[i].label);
}

2. External Audio Tracks (Not Supported)

Video.js does not support externally loaded audio files. Workarounds include playing external audio manually or using Web Audio API, but HLS-embedded tracks are recommended.

3. Built-in Subtitles

HLS streams with embedded subtitles are auto-detected by Video.js. Use player.textTracks() to list available options.

var tracks = player.textTracks();
for (let i = 0; i < tracks.length; i++) {
    console.log(tracks[i].label);
}

4. External Subtitles

External VTT subtitles can be added using the <track> tag inside the video element.

<video controls>
    <source src="video-url.m3u8" type="application/x-mpegURL">
    <track kind="subtitles" src="subtitles.vtt" srclang="en" label="English" default>
</video>

This method enables multilingual support without modifying video files.
